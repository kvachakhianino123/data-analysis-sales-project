# 📊 Sales & Customer Analysis Dashboard

## 🔍 Project Overview
This project analyzes sales and customer data using SQL and Power BI to provide actionable business insights.

The dashboard focuses on:
- Sales performance
- Customer behavior
- RFM segmentation
- Payment methods and order analysis

---

## 🛠 Tools Used
- SQL (Data analysis & querying)
- Power BI (Data visualization & dashboard creation)

---

## 📊 Key Features

### 1. Sales Analysis
- Total sales: 2.23M
- Total orders: 884
- Average order value: 2.52K
- Monthly sales trends
- Country-wise sales performance

### 2. Order Analysis
- Order status distribution (Positive, Negative, Intermediate)
- Top payment methods
- Negative rate analysis

### 3. Customer Analysis (RFM)
- Customer segmentation based on:
  - Recency
  - Frequency
  - Monetary value
- Identified customer groups:
  - Champions
  - Loyal customers
  - At Risk customers
  - New customers
  - Potential loyalists

---

## 📈 Key Insights

- Over 60% of customers fall into "At Risk" and "New" segments  
- Significant opportunity to improve customer retention  
- Certain regions and payment methods show stronger performance  
- Customer segmentation helps target marketing strategies more effectively  

---

## 💡 Recommendations

- Focus on retention strategies for "At Risk" customers  
- Increase engagement with new customers  
- Optimize campaigns for high-value customers (Champions)  
- Analyze negative orders to reduce losses
